# 🏥 Smart Electronic Health Record (EHR) System

An AI-powered **Smart Electronic Health Record (EHR) System** developed as part of the **Advanced Skill Enhancement Project (ASEP)** at **Vishwakarma Institute of Technology (VIT), Pune**.

This system digitizes patient medical records, streamlines doctor consultations, securely stores healthcare data, and leverages a locally hosted Large Language Model (LLM) to generate intelligent clinical summaries for faster patient history review.

---

# 📖 Project Overview

The Smart Electronic Health Record (EHR) System is designed to replace traditional paper-based medical records with a centralized digital platform. It enables healthcare professionals to efficiently manage patient information, consultation records, prescriptions, and medical history while improving clinical decision-making through AI-assisted summaries.

The application provides separate functionalities for doctors and patients, allowing secure storage and quick retrieval of healthcare information.

# 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/uttarwarsuyash-w/Smart-Electronic-Health-Record-EHR-System.git
```

### Navigate to the Project Folder

```bash
cd Smart-Electronic-Health-Record-EHR-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:8000
```
